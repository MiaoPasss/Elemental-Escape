# Elemental-Escape
2D game made with Unity
* Personal project started in August 2021
* Planning to upload to Steam by January 2022

<br/>

In game objects:
* Elements:
    * Fire -> Orange
    * WaterDrop -> Blue
    * Wood -> Brown
    * Iron -> White
    * Coal -> Black

* Collectible Items:
    * Diamond -> Colors corresponding to 'Elements'
    * Coin -> Golden coins can be collected for cosmetics
